# mutual_combine_img
交互にピクセルを置いて、画像を重ね合わせるものです。変わった画像が出力されます。学術的な価値はありません。

画像の(0,0), (0,2),(0,4)...(1,1),(1,3),(1,5)...は画像1の要素

画像の(0,1), (0,3),(0,5)...(1,0),(1,2),(1,4)...は画像2の要素
